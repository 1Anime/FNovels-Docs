# Supported File Extensions

FNovels supports the following file formats for reading novels:

<Frame>
  <CardGrid>
    <Card icon="mdi:file-epub-box" title="EPUB">
      EPUB is the primary supported format. Enjoy a seamless reading experience with bookmarks, progress tracking, and customization options.
    </Card>
    <Card icon="mdi:file-pdf-box" title="PDF">
      PDF files are also supported, so you can read your favorite novels in a familiar format.
    </Card>
  </CardGrid>
</Frame>

<Note>
  More formats may be supported in the future. For now, please use EPUB or PDF files for the best experience.
</Note>
